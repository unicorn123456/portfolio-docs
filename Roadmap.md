| Item | Status | Description |
|---|---|---|

## Present — Hardening & expansion (June 2026)

| Item | Status | Description |
|---|---|---|
| Per-clinic data isolation | In progress | Row-level security policies ensuring no cross-clinic data leakage — a direct GDPR requirement |
| Arabic language support | In progress | RTL UI and Arabic AI triage — unlocks Stockholm's Arabic-speaking patients and the Gulf expansion |
| 30-day trial gate | In progress | Self-serve trial-to-paid conversion flow, removing the manual onboarding bottleneck |
| Conversation history & session grouping | In progress | Lets clinic staff audit what the AI told patients — addresses the trust gap surfaced in user research |

## Strategic throughline

Every near-term feature (RLS, Arabic support, trial gate) directly unblocks either GDPR compliance, the first paying customer, or the Gulf expansion. Nothing on this roadmap is built speculatively.
